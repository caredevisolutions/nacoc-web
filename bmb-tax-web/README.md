# BMB Tax and Financial Service Website

## Overview
A professional, modern, and responsive website for BMB Tax and Financial Service. Built with React, Vite, and Tailwind CSS. The design focuses on trust, clarity, and conversion, using a "Financial Professional" color palette (Deep Blue & Gold).

## Project Structure
- `src/components`: Reusable UI sections (Hero, Services, Contact, etc.)
- `src/App.jsx`: Main layout
- `tailwind.config.js`: Custom theme configuration

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```

## Customization
- **Logo/Branding**: Update `src/components/Navbar.jsx` and `src/components/Footer.jsx`
- **Contact Details**: Update `src/components/Contact.jsx` with actual phone/email/address.
- **Images**: Stock images are currently used. Replace with actual business photos in `src/assets` or updated URLs.
